# references/

외부 참고 자료 모음 — 논문·공식 문서·블로그·치트시트·벤더 가이드 등.
`guides/`의 내 정리 문서와 구분한다: 여기는 **원저자가 내가 아닌** 자료의 링크·발췌·요약.

## 분류 예시 (채워가며 확장)

- `ib/` — InfiniBand/RDMA 스펙, Mellanox/NVIDIA 공식 문서
- `k8s/` — KEP, Kubernetes 공식 문서 스냅샷
- `ml/` — 분산 학습 논문 (Horovod, PyTorch DDP, FSDP, ZeRO)
- `ceph/` — Ceph 논문, Red Hat 운영 가이드
- `papers/` — 일반 시스템/네트워킹 논문

## 정리 원칙

- 링크만 모은 북마크 말고 **왜 읽었는지·어느 섹션이 유용한지** 한두 줄 메모를 붙인다.
- 내 해석이 들어가면 `guides/` 쪽으로 옮기고 여기서는 원문 링크만 남긴다.
